# Конструктор HTML E-mail

Конструктор электронных писем для маркетинговых рассылок. Собран на Parcel + ReactJS + TypeScript.

## Установка

Склонируйте репозиторий и установите зависимости.

```bash
npm install
```

## Запуск локального сервера

```bash
npm start
```
